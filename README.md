# FDAP - Flight, Datafusion, Arrow, Parquet.

- This repository contains projects on the FDAP stack and its various uses across the new composable database architectures.

## Apache Flight
- This is a data transfer network protocol that utilises the Apache Arrow and its IPC format for efficient transfer and is built on gRPC and protocol buffers.

## Datafusion
- This is a data analytics engine built in Rust and utilizing the Apache Arrow ecosystem for high performance.
  
## Apache Arrow
- This is an in-memory data format ideal for analytical workloads.
  
## Apache Parquet
- This is a data storage disk format, ideal for low space usage and interoperability.
